## Features

* Perform basic arithmetic operations

  * Addition
  * Subtraction
  * Multiplication
  * Division
* Clear and Backspace functionality
* Decimal input support
* Keyboard input support
* Button press animations for better user experience
* Display designed to handle long expressions without overflow
* Responsive layout for different screen sizes

## How It Works

The calculator builds mathematical expressions based on user input from buttons or keyboard events. These expressions are evaluated and displayed instantly. Event listeners are used to manage button clicks, keyboard input, and UI updates.
